📊 Sales Data Analysis
📌 Overview

This project analyzes a sales dataset to uncover key business insights such as revenue trends, top customers, and product performance. The goal is to demonstrate practical data analysis using SQL.

🛠 Tools Used

- SQL
- Microsoft Excel
- Power BI

🔍 Key Analysis

- Total revenue calculation

- Top customers by total spending

- Best-selling products

- Sales trends over time

📈 Insights

The analysis helps identify high-value customers, top-performing products, and sales patterns that can support data-driven business decisions.

🚀 Skills Demonstrated

- Data Cleaning

- SQL Queries

- Data Aggregation

- Business Insight Generation
